class LoquaciousDateTimeParserBucket extends DateTimeParserBucket

Overrides DateTimeParserBucket, allowing us to see what fields are set by org.joda.time.format.DateTimeFormatter

WARNING: mutable and non thread-safe. Also, this class does not reset the list of field types after parsing, so the list of field types is good for one parse only.
